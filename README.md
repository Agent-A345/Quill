## ✒️ Quill — Offline Markdown Editor

A powerful, offline-first Markdown editor built using pure HTML, CSS, and JavaScript, featuring a custom Markdown parser, live preview, syntax highlighting, and secure rendering — all without external libraries.

## 🚀 Features
* **Live Markdown Editor** – Write Markdown with real-time preview and smooth rendering.
* **Multi-Document System** – Create, switch, rename, and delete multiple documents.
* **Auto-Save Functionality** – Automatically saves documents using browser localStorage.
* **Custom Markdown Parser** – Supports headings, lists, tables, code blocks, links, images, and more.
* **Syntax Highlighting Engine** – Highlights code for languages like JavaScript, Python, HTML, CSS, and Bash.
* **XSS Protection System** – Sanitizes HTML output to prevent malicious script execution.
* **Scroll Synchronization** – Keeps editor and preview perfectly aligned while scrolling.
* **Toolbar Formatting Tools** – Apply formatting like bold, italic, lists, and code with one click.
* **Export Options** – Download content as Markdown, HTML, or export as PDF.

## 🎮 How to Use
**Writing & Editing**
* Start typing Markdown in the editor
* Preview updates in real-time on the right

**Manage Documents**
* Click + button to create a new document
* Switch between documents from sidebar
* Double-click to rename a document
* Delete documents using the delete icon

**Formatting**
* Use toolbar buttons for headings, bold, italic, lists, etc.
* Use keyboard shortcuts (Ctrl+B, Ctrl+I, etc.)

**Export Content**
* Export as .md or .html
* Print or save as PDF

**Customize UI**
* Toggle between dark and light theme
* Resize editor and preview panes
